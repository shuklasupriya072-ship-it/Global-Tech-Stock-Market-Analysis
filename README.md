
# 📈 Global Tech Stock Market Analysis (2006–2020)
## 🎯 Project Overview
End-to-end analysis of **98 S&P 500 Technology Companies** covering **272,618 rows** of historical stock data from 2006 to 2020.

## ❓ Business Problem
Which tech companies truly created long-term wealth — and why do some thrive in crisis while others collapse?

## 💡 Key Findings
- 🚀 **SHOP** delivered **+5,714%** return over 14 years — highest wealth creator
- 📈 **NVDA** gave **+9,772%** return from lowest to highest price
- 📉 **CAJ (Canon)** was the worst performer — traditional hardware struggled
- 🔵 Cloud & Security companies massively outperformed traditional tech
- 📊 **2020** was the best year for majority of tech companies despite COVID
